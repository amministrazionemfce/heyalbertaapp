# App constants

| File            | Purpose                                                                 |
|-----------------|-------------------------------------------------------------------------|
| `routes.js`     | Pathnames (`ROUTES`), param patterns (`ROUTE_PATTERNS`), URL helpers    |
| `navigation.js` | `MAIN_NAV_LINKS`, footer link columns, footer section headings          |
| `site.js`       | Contact info, footer tagline, copyright & legal link labels             |
| `index.js`      | Re-exports — import from `../constants`                                 |

When you add or rename a route, update **`routes.js`** and **`App.js`** together.
